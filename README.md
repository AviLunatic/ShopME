#E-Commerce Sales Analysis Dashboard
Project Overview

This project focuses on analyzing an e-commerce dataset containing 50,000+ transactions to extract meaningful insights related to sales performance, customer behavior, and business trends.

The analysis was performed using Python, and the interactive dashboard was built using Power BI.

🛠️ Tools & Technologies
Python (Pandas, Matplotlib, Seaborn)
Power BI
Excel / CSV Dataset

📈 Key Features
Interactive Power BI Dashboard
KPI Metrics:
Total Revenue
Total Orders
Total Profit
Average Order Value
Average Shipping Days
Filters (Slicers):
City
Category
Payment Method
Gender
Order Status
Date

🔍 Key Insights
Generated over 1.3 Billion in total revenue
Metro cities like Mumbai and Delhi contribute the highest sales
Clothing is the most sold product category
Customers prefer digital payment methods over cash
Majority of purchases come from young and middle-aged customers
Average delivery time is around 5 days
A small group of customers contributes significantly to revenue (Pareto Principle)
